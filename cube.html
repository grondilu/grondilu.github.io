Below is a colored cube. It's the same that can be found on this this WebGL tutorial.

You should be able to change the view with your mouse.

Apparently your browser does not support HTML5 canvas.

I've changed the code with the goal of implementing Norman Wildberger's idea of Rational trigonometry. Thus there is no trigonometric function (no Math.cos nor Math.sin) and no Math.sqrt call whatsoever.

It does not behave exactly quite as expected yet, but I thought it was close enough to be worth showing.
